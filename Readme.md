# Welcome to `jsx-dotnet`!

The repo aims to start working with JSX and ASP .NET Core easily. We are using [ReactJS.NET](https://reactjs.net/) 

## Installation
    git pull git@github.com:damienasny/jsx-dotnet.git my-project
    cd my-project
    npm install
    dotnet run
